# 讀書計畫

## 近程（率取到開學）
1. 學校學業：完成高三下學業，其中數學的微積分是大學理工科必備的知識，所以我會提前預習並精熟。
2. 英文能力：考取全民英檢中高級或多益 780 分以上，提昇英文能力，使學習、溝通更順暢無礙。
3. 程式方面：在機器學習、AI 更加身加廣學習，其中我比較感興趣的是視覺辨識，這項技術已經應用在很多領域了。
4. 擴展視野：參加各種技術研討會，閱讀期刊、雜誌級學術論文。

2. 大二

4. 大四
